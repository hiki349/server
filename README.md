# How Start Project

- Clone repository from github
```sh
git clone https://github.com/hiki349/server
```

- Start database
```sh
docker-compose up
```

- Install Go
for Mac OS
```sh
brew install go
```

for Linux
```sh
sudo pacman -S go
```

- Start server
```sh
go run cmd/main/main.go
```

## Final your server start on
```sh
http://localhost:3000/
```
